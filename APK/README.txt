Instructions for installing APK: 

1. Go into Settings and make sure Unknown sources is turned on or checked. Some versions of Android may require Verify Apps to be unchecked. (Usually under security settings)
2. Make sure you stop any type of screen filtering apps or widgets that layer over the screen or else it may prevent you from installing it. 
3. Download the APK to your phone.
4. Install it by tapping it and tapping Install. 
5. Turn off Unknown sources in the settings.
